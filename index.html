# Scala for Humans "The useful part" -this presentation is under construction-
Useful part considering a specific target audience Useful part does not imply that the rest is useless
## Who am I
## Humans * Most of us believe to speak a single language even when already does multiple * Mother language, body language, social circle variations and so on * Tend to believe to be more human than others
## Humans * Tend to attach feelings and flags even to inanimate objects and ideas, including programming languages * Are the reason to exist of many solutions but are sometimes forgotten by other humans who develop them * Maybe more on those topics in a future presentation
## What is Scala and why * Programming language created in 2004 * Similar to many other programming languages * Can be used to [introduce children to programming](http://www.kogics.net/kojo)
## What can we do with it * Desktop applications * Mobile apps * Websites * Someday even play with DNA * Actually there are libraries for that [now](https://github.com/bioscala/bioscala/blob/master/src/test/scala/bio/sequence/ntsequence_spec.scala)
## How to start using it * Googling around you will find scary stories about learning curves * Examples usually demonstrate very specific features * Most of the times ["Hello world"](http://www.scala-js-fiddle.com/gist/77a3840678d154257ca1/HelloWorld.scala) is too little and ["Foo Bar"](http://www.scala-js-fiddle.com/gist/9443f8e0ecc68d1058ad/Oscilloscope.scala) is too much * Starting from something that is simple, meaningful and working helps a lot
## What do we need * A computer with Internet connection * JDK * Some working piece of code * You can get one [here](https://github.com/oswaldo/react-scalajs-scalatags)
## How to make it work Start it ```shell activator ``` ```shell [playserver] $ ~run ``` After the required downloads are done, [see it in action](http://localhost:9000)
## How do I change it Find the file AppRouter.scala and change line 32 to ```scala Menu("Item list", Items(Item.Info)) ``` Refresh the browser and see check how it looks
